## What is SurveyJale?

SurveyJale is a web-based survey platform built for collecting rich, qualitative feedback. What makes it different is voice — respondents can answer questions by **speaking naturally** instead of typing. Their words appear on screen in real time as they talk, powered by real-time speech-to-text streaming. Questions can also be **read aloud** to respondents with natural-sounding text-to-speech.

Surveys are fully data-driven. Admins create and manage surveys through a built-in dashboard — no code changes needed to add questions, adjust ordering, or configure conditional logic.

---

## Key Features

**For Respondents**
- Answer questions by typing or speaking — voice input is transcribed live as you talk
- Questions can be read aloud with natural text-to-speech
- Say "next question" to hands-free advance through the survey
- Responses auto-save — nothing is lost if the page refreshes
- Works on desktop, tablet, and mobile (including iPhone Safari)
- Dark and light mode support

**For Admins**
- Create surveys by uploading a simple CSV file
- Audio is automatically generated for each question
- View and manage all survey submissions
- Export submission data to CSV
- Share surveys via a simple link
